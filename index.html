如何搭建私有npm包仓库

一、sinopia搭建私有npm仓库
sinopia是一个零配置的私有的带缓存功能的npm包管理工具。不需要配置数据库。

首先，要配置nodejs及npm的环境，然后
1、安装sinopia
npm install -g sinopia

2、启动sinopia
http://localhost:4873/

3、注册账号/登录账号
npm adduser/npm login

4、发布
npm publish

注：要用到的命令
修改 下载地址 npm
npm config set registry http://registry.cnpmjs.org
修改 淘宝镜像
npm config set registry http://registry.npm.taobao.org/
修改 下载仓库为 私有服务器地址
npm config set registry http://localhost:4873/

包删除
npm –force unpublish 包名

更改版本号
npm version -m ““

其中update_type 有三种：
patch增加一位补丁号（比如 1.1.1 -> 1.1.2）
minor增加一位小版本号（比如 1.1.1 -> 1.2.0）
major增加一位大版本号（比如 1.1.1 -> 2.0.0）

发表于 2017-08-07

ES6 Object.assign 将源对象（source）的所有可枚举属性，复制到目标对象（target）

一、基本用法
它至少需要两个对象作为参数, 第一个参数是目标对象，后面的参数都是源对象,只要有一个参数不是对象，就会抛出TypeError错误。

Object.assign(target,source1,source1)

注：如果目标对象与源对象有同名属性，或多个源对象有同名属性，则后面的属性会覆盖前面的属性。
如果该参数不是对象，则会先转成对象
undefined和null无法转成对象，所以如果它们作为参数，就会报错。
如果源对象某个属性的值是对象，那么目标对象拷贝得到的是这个对象的引用。

二、用途

  1. 为对象添加属性
  2. 为对象添加方法
  3. 克隆对象
  4. 合并多个对象
  5. 为属性指定默认值
